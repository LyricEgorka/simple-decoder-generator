Это простенькое приложение для настольной игры Decoder, которое имеет 3 базы слов по уровням сложности, генератор последовательностей шифра и базовую систему подсчёта очков.
This is a simple application for board game "Decoder" that has 3 word bases by difficulty levels, a cipher sequence generator, and a basic scoring system.
